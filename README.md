
# 🕊️ TodoLab : Make Your Day Feel Lighter

**TodoLab** is a minimalist, aesthetic task management web application designed to reduce digital clutter and help you focus on what truly matters. With a soft, modern UI and a "calm-first" philosophy, it turns your to-do list into a stress-free experience.

![Status](https://img.shields.io/badge/status-active-success.svg)

---

## ✨ Features

- **Minimalist Design:** A clean, airy interface with soft gradients and high-readability typography.
- **Fluid Workflow:** Add tasks instantly and "launch your flow" with zero friction.
- **Dynamic Tracking:** Real-time counters for Total, Active, and Completed tasks.
- **Smart Filtering:** Quickly toggle between `All`, `Active`, and `Done` views.
- **Bulk Cleanup:** One-click "Clear Done" to keep your workspace pristine.
- **Personalized Themes:** Built-in theme switcher to match your mood or environment.

## 🚀 Getting Started

### Prerequisites
You only need a modern web browser (Chrome, Firefox, Edge, or Safari).

### Installation
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/your-username/todolab.git](https://github.com/your-username/todolab.git)


2.  **Navigate to the directory:**
    ```bash
    cd todolab
    ```
3.  **Open the app:**
    Simply open `index.html` in your browser.
